# TeleSales 21 - Template approval and campaign deletion

- Adds a visible template approval status card to each campaign.
- Sending WhatsApp campaigns is blocked until a manager confirms the exact 360dialog template is Approved/Active.
- Shows where to check the status in 360dialog.
- Adds a permanent Delete Campaign action with exact-name confirmation.
- Deletion removes campaign recipients, events, follow-ups and uploaded image.
- Opt-out/suppression history is intentionally retained for compliance.
- Existing PostgreSQL databases are upgraded automatically at startup.
