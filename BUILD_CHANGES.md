# TeleSales WhatsApp and Email Callback Build

Added:
- Communications campaign dashboard and campaign creation.
- Selection of existing `LapsedPolicy` leads as recipients.
- Combined WhatsApp and HTML email messages.
- Secure callback, not-interested and opt-out links.
- Automatic callback status updates in the existing recovery queue.
- Assigned-agent notifications with direct client/application actions.
- Online application prefilling from the linked lead.
- Per-contact communication preferences.
- Hashed cellphone/email suppression records for opt-outs.
- WhatsApp webhook verification and response handling endpoint.
- Deployment/environment documentation.

Important deployment settings are documented in `WHATSAPP_EMAIL_CAMPAIGNS.md` and `.env.example`.
