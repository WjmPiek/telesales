TeleSales 23 - Campaign image persistence and template status fixes

1. Campaign advert images are now stored in PostgreSQL instead of Render's temporary filesystem.
2. Existing campaigns with a missing image show a Replace image control.
3. Campaign media has a public read-only URL so WhatsApp can fetch the image header.
4. Existing campaigns can edit the exact template name and language, then run a live check.
5. Template matching accepts common name formatting and English language variants.
6. A successful provider lookup that does not find the requested template returns a normal status response instead of HTTP 422.
7. When a template is not found, the status message lists templates returned by the connected WABA to help select the correct one.

After deploying, open each old campaign with a broken image and use Replace image once.
